# Captcha Solver Frontend This project provides a simple, single-page responsive HTML application designed to act as a frontend for solving visual captchas. It's built with modern web standards, uses Tailwind CSS for styling, and includes basic JavaScript to dynamically load captcha images.## Features*   **Responsive Design:** Adapts to various screen sizes, from mobile to desktop.*   **Dynamic Image Loading:** Can load captcha images from a specified URL parameter or default to a local image.*   **User Input Interface:** Provides a clear input field for users to enter the captcha solution.*   **Tailwind CSS:** Styled with utility-first CSS for a clean and modern look.## Usage1.  **Open `index.html` in your browser.**2.  **Default Captcha:** By default, the application will display `sample.png` as the captcha image. The text for `sample.png` is "ADCRB".3.  **Load Custom Captcha:** To load a captcha from a specific URL, append the `url` query parameter to `index.html`.*   Example: `index.html?url=https://example.com/path/to/your/image.png`4.  **Enter Solution:** Type the text you see in the captcha image into the input field.5.  **Submit:** Click the "Submit" button or press `Enter`. The application will display a message indicating the submitted text. In a real-world scenario, this submission would typically be sent to a backend server for validation.## DevelopmentThis project consists of a single `index.html` file, `README.md`, and `LICENSE`. No complex build steps or dependencies beyond a web browser are required.## Technologies Used*   HTML5*   Tailwind CSS (via CDN)*   JavaScript (Vanilla JS)## LicenseThis project is open-source and available under the MIT License.
